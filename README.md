# Run the projectin development mode
```
yarn
yarn start
```

# Run the project in production mode
```
docker-compose up -d
```

# Project
Rick & Morty App to fetch characters data, episodes and locations.

- React.js
- Typescript
- Code Formatters: ESlint + Prettier + Editor Config
- Ant Design UI
- GraphQL
- Docker + Docker Compose
- CI/CD
- Git Hub Actions
